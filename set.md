和集合  
for all x, (x 要素 途中~) iff (x )


for all x は,どのような x を持ってきても,それ以下の式が成立するという意味  
iff は数学的には " X であるのは Y のとき,かつ,そのときに限る" という表現で, X と Y が等価であることを示す.
英語では " X if and only if Y " あるいは縮めて, " X iff Y " となる
